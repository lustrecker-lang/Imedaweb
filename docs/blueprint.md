# **App Name**: IMEDA

## Core Features:

- HomePage: A HomePage with navigation on top and footer at the bottom. Responsive and modern.
- Admin Login: A login page for page admins to login and change content of the Home page and all future pages.

## Style Guidelines:

- Primary color: White (#FFFFFF) to keep it clean.
- Background color: Sand (#F6F1EB) for a clean and professional look.
- Accent color: Deep Blue (#051E3A) to highlight important information and calls to action.
- Font pairing: 'Unna' (serif) for headlines and use Inter for text.
- Use clear, professional icons if needed
- Clean, grid-based layout with a focus on readability and easy navigation.
- Subtle animations to highlight user interactions.
- Border radius always everywhere 2pixels.
- Don't have hover shadows.
